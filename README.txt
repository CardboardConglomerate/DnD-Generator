D&D Character Generator
Files connected to the Cardboard Conglomerate D&D Character Generator.
The D&D Character Generator created by the Cardboard Conglomerate. Currently Python and PyQt5 are required to run this program.
If you try and click either the Generate Subrace or Generate Subclass buttons without a race or class having already been generated the programm will crash.
